# New Roop Sajja Beauty Parlour — Website

## Files
- `index.html` — the complete website (design, menu, booking, reviews, admin panel — all in one file)

## How to use
1. Open `index.html` in any browser to preview it directly.
2. To put it live on the internet, upload this file to any web hosting (Hostinger, GoDaddy, Netlify, GitHub Pages, etc.) or point a domain (e.g. newroopsajja.com) to it.

## Owner Admin Panel
- Click the ⚙ gear icon (top-right of the site).
- **First time**: no passcode exists yet, so it will ask the owner to set her own passcode (min 4 characters). Only she should do this.
- **After that**: same gear icon asks for that passcode to log in.
- Passcode can be changed anytime from inside the admin panel ("Passcode Badlein").
- Note: since this is a static site with no real server-side authentication, the passcode is stored in the site's shared storage in a lightly obfuscated (not encrypted) form — good enough for keeping casual visitors out, but not bank-grade security. A real login system would need proper backend hosting.

From the admin panel the owner can:
- Post/hide an announcement or offer banner
- Update the shop timing text
- Add/remove holiday dates
- Upload gallery photos
- Add YouTube video links
- View and delete customer reviews
- Change her own passcode

## Customer Reviews
- Anyone visiting the site can post a name + star rating + review text (Reviews section, above the booking area).
- Reviews appear publicly right away.
- Owner can delete any review from the admin panel.

## Clickable links
- Location text and the map preview both open the parlour's Google Maps link in a new tab.
- Instagram and YouTube icons (in the Contact section) open those profiles in a new tab.
